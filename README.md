Телеграм бот, который принимает картинку, убирает у нее задний фон и меняет разжешение на 512х512, чтобы потом ее можно было использовать, как стикер. Работа выполнена студентами Александром Стахиевым 3530904/70103 и Загородним Георгием 3530904/70105

Для запуска необходимо установить зависимости: pip install -r requirements.txt
Запуск производился на бесплатном хостинге Heroku
$ heroku login
$ heroku create
$ git push heroku master
$ heroku ps:scale web=1
$ heroku open

Юнит тетсы запускаются командой: python -m unittest discover
